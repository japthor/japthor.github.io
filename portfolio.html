Skyline Bowling

I developed and shipped new gameplay features and SDK updates, set up and tracked AB tests, and fixed bugs to improve stability. I also worked on performance optimisations to make the game run smoother and reduce build size.

Derailed: Survival Adventure

Hybrid-Casual Survival Adventure Game where players drive a train across dangerous landscapes, gather resources, recruit a crew, and make key decisions to save humanity.

I focused on improving performance and reducing build size to make the game run smoother, while also fixing bugs to increase stability. Alongside this, I implemented new gameplay features, handled SDK updates, and set up and analyzed A/B tests.

Oil Digging 3D

Hybrid-Casual Game where you build and operate an oil rig right from the beginning of finding an oil well and starting to drill to building an efficient pipeline system to transport it.

I had the privilege of being part of the team developing one of the company’s first hybrid-casual games, marking a new direction after years of focusing on hyper-casual titles. I was involved throughout the entire development cycle, from prototype to final product. My responsibilities included coding the logic for various elements such as the machines, resource collection and delivery systems, and the in-game shop, among others.

Drawer Sort

I was involved in the full development cycle, from prototype to final product. My responsibilities included coding key gameplay logic, such as object movement, streamlining level creation for designers to make the process faster and more efficient, and implementing the hint system, among other features.

Tower Smash

Hybrid-Casual Game where players need to break, explode, and blast their way to victory as they aim, shoot, and fire colorful balls at the towering structure. Their mission? Bring that tower down!

Helix Jump

Hyper-casual game where the player rotates a helix tower to guide a bouncing ball downward through gaps, avoiding obstacles. It features simple one-touch controls, fast-paced gameplay, and short, satisfying sessions.

I worked on implementing and maintaining new gameplay features while managing SDK updates to ensure the game stayed up to date. In parallel, I set up and monitored AB tests to evaluate changes and identify what worked best. A big part of my focus was fixing bugs and improving stability, as well as optimising performance to make the game run more smoothly and reduce overall build size.

Sharpshooter Blitz

I focused on fixing bugs and improving the game’s performance to ensure smoother gameplay, while also adding new gameplay updates to keep the experience fresh. In addition, I managed SDK updates to maintain compatibility and support ongoing development.

Baseball Boy

Hyper-casual game where players hit balls to earn coins and upgrade their bat, power, and abilities. It features simple tap controls, addictive progression, and short, satisfying gameplay loops focused on improving scores and reaching new milestones.

I spent a lot of time fixing bugs and improving the game’s stability, while also working on performance optimisations to make it run smoother and reduce build size. At the same time, I added and maintained new gameplay features and handled SDK updates to keep the game current. I also set up and tracked AB tests to see which changes performed best and guide future updates.

Hospital Simulator 3D
